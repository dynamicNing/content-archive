## 第二步：公司基本面速写

### 一句话定位

**蔚来是中国高端智能电动汽车制造商**，核心定位"高端换电智能电动车"，对标BBA（奔驰/宝马/奥迪）价位段，同时运营电池换电站网络（NIO Power）和用户社区（NIO House）。

### 发展简史

| 时间 | 事件 |
|------|------|
| 2014年 | 蔚来汽车成立，创始人为李斌（原易车网创始人） |
| 2017年 | 首款量产车ES8发布 |
| 2018年 | 赴美IPO（NYSE: NIO） |
| 2022年 | 港股双重上市（9866.HK） |
| 2024年 | 子品牌ONVO（乐道）发布，聚焦家庭市场 |
| 2025年Q4 | **首次实现GAAP季度盈利** |
| 2026年4月 | 旗舰ES9上市，三品牌同台 |

### 业务结构（引用最新财报数据）

**注**：港股财务数据获取受限，以下数据主要来源于美股财报（CNBC数据）及研究机构引用：

| 业务板块 | 概况 |
|---------|------|
| **整车销售** | 核心收入来源，2025年销量约22万辆（同比+40%+） |
| **电池即服务（BaaS）** | 可租可换电模式，降低购车门槛 |
| **NIO Power** | 换电站网络，已建成数千座，开放给其他品牌 |
| **NIO Life** | 周边商品/生活方式品牌 |
| **子品牌** | ONVO（家庭市场），firefly（入门级） |

### 产业链位置

- **上游**：电池采购（宁德时代等），芯片（英伟达合作）
- **下游**：直接2C销售，无传统4S店
- **壁垒**：换电网络（重资产护城河）、高端品牌形象、用户社区粘性

3. **用户忠诚度**：NIO App社区活跃，续保率高于行业均值

**护城河威胁**：
- 华为问界M9在50万+价位直接竞争
- 极氪、小米SU7 Ultra等新进入者
- 换电模式被质疑（充电速度提升使换电优势缩小）

**核心假设**：2026年交付增长40-50%，ES9月销过5000辆，BaaS渗透率提升

### 4.5 新业务/转型
