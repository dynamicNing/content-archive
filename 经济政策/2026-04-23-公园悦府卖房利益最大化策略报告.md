## 一、核心结论（先说重点）

> 在当前市场下，**不装修裸卖的成交价约370~376万**，而**经过专业轻装修+ staging（房屋布置）后，成交价可达385~400万**，溢价约 **10~25万**，投入产出比约 **1:3~1:5**。

**最佳策略组合：**
```
轻装修（3~5万）+ 专业staging（0.5~1万）+ 经纪人选择（关键！）
+ 上市时机（重要！）= 潜在溢价15~30万
```

## 二、房屋现状诊断与改造优先级

### 2.1 你的房子当前痛点（81㎡户型常见问题）

| 痛点 | 影响买家决策程度 | 解决难度 | 优先级 |
|------|---------------|---------|--------|
| 厨房/卫生间老旧感 | ⭐⭐⭐⭐⭐ | ★★☆ | 🔴 最高 |
| 墙面污渍/裂纹 | ⭐⭐⭐⭐ | ★☆☆ | 🔴 高 |
| 灯光昏暗（缺乏分区照明）| ⭐⭐⭐ | ★☆☆ | 🟡 中 |

### 2.2 轻装修清单（预算3~5万）

#### 🔴 必须做（投入≈2.5万，溢价回报最高）

**① 厨房台面翻新（重中之重）**
- 现状：2012~2016年交付的厨房台面多为人造石，染色、磨损严重
- 方案：更换石英石台面（约2米），或使用石材清洁剂+釉面修复
- 花费：约 3,000~6,000元
- 效果：厨房是买家最关注的区域，翻新后视觉冲击最强

**② 卫生间全套翻新**
- 重点：马桶/洗手盆/淋浴五金件+墙面地面清洁
- 花洒、龙头更换（现代简约风格，约1,000~2,000元/套）
- 花费：约 5,000~10,000元

**③ 全屋墙面处理**
- 步骤：墙面打磨 → 底漆 → 两遍面漆（白色/浅灰）
- 花费：约 5,000~8,000元（81㎡套内面积）
- 注意：选白色或浅灰，不要彩色（影响采光和买家想象空间）

**④ 灯具更换**
- 现状：开发商原始灯具普遍老旧，色温偏暖/偏暗
- 方案：客厅换筒灯/射灯（4000K中性光），卧室换简约吸顶灯
- 花费：约 1,500~3,000元

#### 🟡 建议做（投入≈1.5万）

**⑤ 地板清洁/打蜡**
- 实木地板：深度清洁+打蜡，花费约 800~1,500元
- 复合地板：如有磨损，考虑局部更换（约3,000~5,000元）

**⑥ 门窗/踢脚线翻新**
- 踢脚线老化：更换PVC或木塑踢脚线，约 500~1,000元
- 门皮脱落：贴波音片（免漆纸，约500~1,000元/扇）

**⑦ 全屋开关插座面板更换**
- 换成无边框简约款（约20~40元/个，共约10个）
- 花费：约 300~800元

#### 🟢 可选（投入≈5,000元）

**⑧ 窗帘更换**
- 选白色透光纱帘+深色遮光帘，简约现代感
- 花费：约 1,500~3,000元

**⑨ 阳台功能化改造**
- 拆除旧晾衣架，改为绿植/休闲角（买家的"生活方式想象"）
- 花费：约 1,000~2,000元

## 三、专业 Staging（房屋布置）策略

### 3.1 Staging核心原则

```
卖家视角：我在展示一个"我曾经的家"
买家视角：他们看到的是"我未来的家"
核心：让买家产生情感代入，而不是看到"别人的旧房子"
```

### 3.2 具体操作（可DIY，无需专业人士）

| 操作 | 说明 | 花费 |
|------|------|------|
| **客厅去杂物** | 移除个人物品、玩具、堆积杂物 | 0元 |
| **沙发套更换** | 换浅色简约款，遮盖旧沙发 | 300~800元 |
| **床上用品** | 换白色/浅灰纯棉，卧室立刻高级感 | 300~600元 |
| **绿植点缀** | 客厅放1~2盆中等尺寸绿植（龟背竹/琴叶榕）| 200~500元 |
| **挂画** | 客厅挂1~2幅简约风格装饰画 | 300~800元 |
| **香薰** | 选用中性香（白茶/木质），看房前15分钟点香 | 50~100元 |
| **全屋镜面** | 玄关/客厅放大镜面，空间显大 | 200~500元 |
| **餐桌布置** | 放一束鲜花+餐垫，立刻有生活气息 | 100~300元 |

> 💡 **关键技巧：** 买家看房时，家里**永远不要有人**（卖家在场会让买家有压迫感）

## 四、经纪人选择策略（极其重要！）

### 4.1 好经纪人标准

| 维度 | 好经纪人 | 差经纪人 |
|------|---------|---------|
| 响应速度 | 1小时内回复 | 1~2天不回 |
| 市场分析 | 有数据支撑，逻辑清晰 | 空话 |
| 推房策略 | 全渠道投放，精准匹配客户 | 只挂贝壳等平台 |
| 议价能力 | 帮卖家守住底价 | 急于成交，劝你降价 |
| 配合度 | 随时可约看房 | 周末才能看 |

### 4.2 佣金谈判

> 北京二手房买卖佣金通常为房屋成交价的1.5~2.5%，**可以谈！**

| 策略 | 说明 |
|------|------|
| "多家比选" | 告知你在比较3个经纪人，佣金有竞争空间 |
| "我配合看房" | 承诺随时可看、配合度高，可要求折扣 |
| 底线 | 佣金不低于1%，否则经纪人缺乏推房动力 |

## 五、定价与时机策略

### 5.1 定价区间

```
最低心理底价：355~360万（可谈空间）
合理挂牌价：370~378万
理想成交价：365~375万
```

> 💡 第一次挂牌**不宜过高**。高价挂牌后持续无人问津，再降价的房子，买家会认为"有问题"。

### 5.2 最佳卖房时机

| 时机 | 原因 | 效果 |
|------|------|------|
| **每年3~4月（春季）** | 买家开始看房，学区需求释放 | ⭐⭐⭐⭐⭐ |
| **每年9~10月（秋季）** | 930政策窗口，买家观望情绪最低 | ⭐⭐⭐⭐⭐ |
| 政策降息后1~2个月 | 买家预期改善，入市意愿增强 | ⭐⭐⭐⭐ |
